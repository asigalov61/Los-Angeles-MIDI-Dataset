## Main Features:

### 1) ~405000 100% unique MIDIs to explore :)
### 2) Each MIDI file was read-checked and 100% de-duped
### 3) Extensive meta-data for each MIDI file
### 4) Full chords data for each MIDI file
### 5) Helper Python code
